What is a Website?

a website is a collection of interconnected files.       It usually consists of:
  • one or more webpages
  • media content
  • technical resources (style sheets or scripts)
  • It lives on a server, which is a computer that is always connected to the internet.
  • The first website was info.cern.ch
  • the first web-browser looked like this
webpages? stylesheets? scripts?
A webpage is a hypertext document
  • Hypertext is text which contains links to other texts.
  • It often refers to other files: It tells the browser where they are and where/how to display them.
A stylesheet defines how webpage-elements look
  • it is written in CSS (the web's style language)
  • it defines rules for the display of a webpages elements
  • it can be inside of a webpage, or exist as a seperate document
a script is a program which can enhance the functionality of a webpage this resource doesn't cover scripts, because they're kind of complicated and not necessary for creating websites.
